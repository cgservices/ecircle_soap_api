# e-Circle Api
A Client for e-Circle Api communication

## Installation
### In Gemfile
    gem 'ecircle_api', :git => "git://github.com/cgservices/dcgw_api.git"

## Usage
    @dcgw_client = DcgwApi::Client.new(url_to_ecircle_rest_api), options={})

###options:
    username: username
    password: password
